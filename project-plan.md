## Project Overview

Build a CNN-based classifier that detects whether individual parking spaces are **occupied** or **empty** from overhead parking lot images.

**Approach:**
1. Load a labeled parking lot dataset (Kaggle parking space detection)
2. Crop individual parking spots from lot images using XML annotations
3. Train a CNN to classify each spot as occupied or empty
4. Evaluate accuracy and visualize results on full lot images with color-coded overlays

# PART 1 — Data & Preprocessing
### Assigned to: **Kellen Andrews**
**Files:** `preprocess.py`

### Steps
1. Parse `annotations.xml` to get polygon coordinates and labels for each parking spot
2. For each polygon, compute bounding box (min/max x and y)
3. Crop that bounding box from the full lot image using OpenCV
4. Save crop to `data/empty/` or `data/occupied/` based on label
5. Verify output counts — run `ls data/empty | wc -l` and `ls data/occupied | wc -l`
6. Document dataset stats for the report (total images, class balance, image size)

**Labels in XML:**
- `free_parking_space` → save to `data/empty/`
- `not_free_parking_space` → save to `data/occupied/`
- `partially_free_parking_space` → treat as occupied

---

# PART 2 — Model Architecture & Training
### Assigned to: **Kellen Andrews**
**Files:** `model.py`, `train.py`

### Steps
1. Define model in `model.py` — use pretrained **ResNet-18** with final layer replaced for binary classification (2 outputs: empty / occupied)
2. In `train.py`, load cropped images from `data/` using `torchvision.datasets.ImageFolder`
3. Apply transforms: resize to 64x64, normalize with ImageNet mean/std
4. Split dataset 80/20 train/val
5. Train with CrossEntropyLoss + Adam optimizer for 10 epochs
6. Save best model weights to `best_model.pth`
7. Target val accuracy: **90%+**

---

# PART 3 — Evaluation & Demo
### Assigned to: **JJ Araim**
**Files:** `evaluate.py`, `demo.py`

### Step 3.1 — Evaluate (`evaluate.py`)
1. Load `best_model.pth`
2. Run model on full dataset
3. Generate classification report (precision, recall, F1)
4. Generate and save confusion matrix as `confusion_matrix.png`

### Step 3.2 — Visual Demo (`demo.py`)
1. Load a full lot image and its annotations
2. For each annotated spot, crop it and run the model
3. Draw the polygon outline on the full image — **green = empty, red = occupied**
4. Overlay a text summary (e.g. "Empty: 12  Occupied: 20")
5. Save output as `demo_output.png` — this is what gets shown during the presentation
